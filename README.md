# Sugarfreeify

Transform a project with sugar syntax to sugarfree syntax

## Usage


```
  Usage
    $ sugarfreeify [inputextension] <processor> [outputextension]

  <processor> is one of:
    - sugarml
    - sugarss

  Examples
    $ sugarfreeify sugarml
```
